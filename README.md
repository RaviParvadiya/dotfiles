This repository contains my personal configuration files for Linux, including:

- **Zsh** (with plugins, completions, aliases, and Starship prompt)
- **Neovim** (Lua-based, with LSP, formatting, and plugin management)
- **tmux** (custom keybindings, theme, and plugin manager)
- **Hyprland** (Wayland compositor, keybinds, and autostart apps)
- **Other tools** (fzf, zoxide, etc.)

## Features

- Modern, modular Neovim setup with LSP, DAP, formatters, and linters
- Productive Zsh shell with autosuggestions, syntax highlighting, and fuzzy completion
- Tmux with Vim-style navigation and session management
- Hyprland config for a tiling Wayland desktop
- Language-specific indentation and formatting rules

## Installation

1. Clone this repo:
   ```sh
   git clone https://github.com/RaviParvadiya/dotfiles.git ~/dotfiles
   ```
2. Symlink configs or copy them to your home directory as needed.
3. Install required tools (see below).

## Requirements

- [zsh](https://www.zsh.org/)
- [starship](https://starship.rs/)
- [neovim](https://neovim.io/) (v0.9+ recommended)
- [tmux](https://github.com/tmux/tmux)
- [fzf](https://github.com/junegunn/fzf)
- [zoxide](https://github.com/ajeetdsouza/zoxide)
- [Hyprland](https://hyprland.org/) (Wayland only)
- [exa](https://github.com/ogham/exa) (optional, for better ls)
- [ripgrep](https://github.com/BurntSushi/ripgrep) (optional, for search)
- [node, npm, python, go, php, etc.] for language tooling
